### [Valunds ServiceBok](https://valunds.se)
Digital service book SaaS for vehicle owners and businesses. Keeps documentation centralized, builds trustworthy service histories, and generates QR-verifiable value certificates.
*   **Stack:** Django, HTMX, DRF, React Native (Expo), Stripe, Cloudinary, OpenAI.
*   **Focus:** Security-focused architecture, Apple/Google Play compliant deletion flows, and safe media handling.

### [SkogsKvitto](https://skogskvitto.se)
Focused receipt and documentation platform tailored for forestry and agriculture.
*   **Stack:** Django, HTMX.
*   **Focus:** Niche workflows, property-aware structures, strict access control, and accountant/revisor readiness.

### [La Serenity](https://laserinity.fr)
Multilingual wellness platform.
*   **Stack:** Django, Wagtail CMS, React, TypeScript, Docker, Fly.io.

### sieverk (Side Project)
A Rust sidecar project exploring SIE file parsing, tokenization, and validation to build future accountant-friendly reporting engines for SkogsKvitto.

## Engineering Principles

> _Build real systems. Protect the data. Keep the architecture clean._

*   **Security First:** Access control, rate limiting, and safe defaults are part of the architecture, not an afterthought.
*   **Privacy by Design:** Permanent deletion pipelines with vendor cleanup, respecting user data lifecycles.
*   **Production Readiness:** Reliable CI/CD deployments and documentation-driven development.
